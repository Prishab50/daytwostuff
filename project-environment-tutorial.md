# Step-by-Step Tutorial to Creating a New Project Environment

### 1) Create a New Github Repository
* Initialize it with a README.md, license, and .gitignore file
* In said repository, create a new codespace
* Open the code space in your local VSCode environment

### 2) Prep the Codespace
* Create a virtual environment
* Add the virtual environment to the .gitignore file
* Create a requirements.txt file
* Add all necessary packages to the file
* Activate the virtual requirement and pip install the requirements.txt file to it

### 3) Starting the Project
* Create a new .py file
* Install the jupyter extension/ipykernel
* Install the Data Wrangler extension
* Import the necessary packages into the python file
* Find and load the dataset into the file 
* View the dataset using data wrangler by clicking on jupyter and the variable button

### 4) Update the Repository
* Add and push all changes made to the repository
